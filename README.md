# semibasement-web-v2

semibasement 의 새로운 버전

> react, redux, typescript

## Dev Enviroment

- VS Code
- macOS
- npm 6.7.0
- yarn 1.13.0

## Configuration

### VScode 설정 (settings.json)

```json
// ...
"editor.formatOnSave": true,
"eslint.autoFixOnSave": true,
"eslint.validate": [
    "javascript",
    "javascriptreact",
    { "language": "typescript", "autoFix": true },
    { "language": "typescriptreact", "autoFix": true }
]
```

## Install

```sh
yarn install
```

## Command Script

### Dev

```sh
yarn dev
```

### Start

```sh
yarn start
```

### Build

```sh
yarn build
```
